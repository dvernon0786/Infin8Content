# Infin8Content - Project Documentation Index

Welcome to the comprehensive documentation for **Infin8Content**, a platform designed for automated high-quality content generation with AI grounding and SEO optimization.

## 🚀 Quick Reference

- **Primary Stack**: Next.js 16.1.1, React 19.2.3, TypeScript 5
- **Database**: Supabase (PostgreSQL) with Row-Level Security (RLS)
- **Engine**: Inngest-powered workflow automation
- **AI Integration**: OpenRouter (LLM) and Tavily (Web Grounding)
- **Infrastructure**: Vercel-hosted, Sentry monitoring, Stripe payments
